# 레일플러스 사용자 홈페이지(반응형)

## 📌 프로젝트 소개

레일플러스 사용자 홈페이지(반응형) 입니다.

## 🛠 기술 스택

- **Frontend**: React v19.0.0, Next.js v15.1.7, javascript
- **스타일링**: Tailwind CSS

## 🔄 시스템 요구사항

- Node.js v18.20.0 이상
- npm
- 최신 버전의 웹 브라우저

## 🚀 시작하기

```bash
# 최초 설치시
npm cache clean --force

# 패키지 설치
npm install --legacy-peer-deps

# 개발 서버 실행
npm run dev

# 프로덕션 빌드
npm run build

# 프로덕션 서버 실행
npm run start

#스토리북 실행
npm run storybook
```
